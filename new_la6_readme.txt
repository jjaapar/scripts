Example usage scenarios:

Basic status check:
python lapoe_controller.py get-status

Interactive mode:
python lapoe_controller.py interactive

Mute control:
python lapoe_controller.py mute 1

LED control with custom values:
python lapoe_controller.py run-control 255 128 64 32 16 8

Smart mode configuration:
python lapoe_controller.py smart-mode 5
